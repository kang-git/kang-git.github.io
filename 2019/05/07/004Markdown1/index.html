Markdown特点

1.Markdown是一种轻量级的标记语言，它允许人们使用易读易写的纯文本格式编写文档。

2.Markdown语言在2004由约翰·格鲁伯（John Gruber）创建。

3.Markdown编写的文档可以导出HTML、Word、图像、PDF、Epub等多种格式的文档。

4.Markdown编写的文档后缀为.md,.markdown。

Markdown编辑器

  • Typora（支持MacOS、Windows、Linux平台，且包含多种主题，支持到处HTML、PDF、Word、图片等多种类型）

2.使用 # 号标记

使用 # 号可以表示1-6级标题，一级标题对应一个 # 号，二级标题对应俩个 # 号，以此类推。

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题
